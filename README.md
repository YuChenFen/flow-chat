## 项目运行
### 安装相关依赖

```bash
npm install
```

### 运行

```bash
npm run dev
```

### 打包构建
- windows
```bash
npm run build:win
```

- macos
```bash
npm run build:mac
```

- linux
```bash
npm run build:linux
```

### 安装
