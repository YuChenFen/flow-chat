# 暂无文档介绍


### 运行

```bash
npm run dev
```

### 打包构建
- windows
```bash
npm run build:win
```

- macos
```bash
npm run build:mac
```

- linux
```bash
npm run build:linux
```

### 安装

版本用于本地构建，在安装之前请到 [flow-chat-serve](https://github.com/YuChenFen/flow-chat-serve)，运行服务端。
